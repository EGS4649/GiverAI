API Documentation

Generate Tweet API

Endpoint: POST /generate-tweet-api

Authentication: API Key in Header

Headers:

API-Key: your_api_key_here

Parameters:

  • job (string) - Your profession/niche
  • goal (string) - Your goal for the tweet

Example Request:

curl -X POST \
  http://yourdomain.com/generate-tweet-api \
  -H 'API-Key: your_api_key_here' \
  -d 'job=Digital Marketer&goal=Promote new blog post'

Successful Response:

{
  "tweet": "Check out my latest blog post on digital marketing trends!"
}

Error Responses:

401 Unauthorized - Invalid API key
403 Forbidden - API access not available for your plan
